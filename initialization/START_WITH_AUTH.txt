         Re-login with credentials

  Issue: "No map dots showing"
  Fix:   CSV must have "region" column
         Re-import data with correct column

  Issue: "Donut chart empty"
  Fix:   CSV must have "category" column
         Use SAMPLE_DATA.csv as example

  Issue: "Cannot login after register"
  Fix:   Wait 2-3 seconds after registration
         Try login again

  Issue: "Import fails"
  Fix:   Check CSV has all 6 columns:
         date, category, region, revenue, quantity, cost
         Restart backend: docker-compose restart api

  Issue: "Database error on start"
  Fix:   Full reset: docker-compose down -v && docker-compose up -d
